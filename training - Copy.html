Algorithm Used

This flood prediction system uses the Random Forest Machine Learning Algorithm. Random Forest builds multiple decision trees using historical environmental data such as:

  • Temperature
  • Maximum Temperature
  • Wind Speed
  • Cloud Cover
  • Precipitation (Rainfall)
  • Humidity

Each tree votes for a prediction and the final result is determined by majority voting. This improves prediction accuracy and reduces overfitting.
